# Image Guidelines
Some general guidelines for the images that Apollo can use and what the UI is expected to work with.

These are essentially recommendations and I would love some feedback on that topic (disagreeing or not).

## Recommendations / Guidelines
In general, Apollo should be able to work with any image size
and prefers high quality/resolution images, over compressed or resized ones.

I recommend **against** adding black borders/bars and not upscaling images to meet minimum size requirements.

| Type     | Aspect ratio (w:h) | Minimum size | Transparency allowed | Notes                                                        |
|----------|--------------------|--------------|----------------------|--------------------------------------------------------------|
| Poster   | 2:3                | 500x750 px   | No                   | A 'clean' poster works best for Apollo (e.g. just the title) |
| Backdrop | 16:9               | 1280x720 px  | No                   | Should not contain spoilers                                  |
